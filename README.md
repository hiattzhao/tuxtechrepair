# TuxTechRepair

**Sustainable Tech for Everyone.**

TuxTechRepair is a non-profit organization dedicated to bridging the digital divide by repairing retired computers and donating them to those in need. This repository contains the source code for our single-page website.

## üöÄ Features

- **Premium Dark Aesthetics**: A modern, high-contrast design featuring glassmorphism elements and smooth gradients.
- **Responsive Navigation**: A fully mobile-responsive header with a toggleable menu.
- **Interactive Sections**:
  - **Mission**: Highlights the e-waste problem and our solution.
  - **Services**: Details our repair and donation services.
  - **Get Involved**: Calls to action for volunteering and donating equipment.
- **Functional Forms**:
  - **Volunteer Sign-Up**: A comprehensive form for prospective volunteers.
  - **Equipment Donation**: A dedicated form for scheduling tech donations.
  - **Email Integration**: Both forms handle submissions via the Fetch API, sending data directly to our team via email without page reloads.
- **Animations**: Subtle fade-in on scroll effects for an engaging user experience.
